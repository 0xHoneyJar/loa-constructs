# Design Direction

## We Want

| Attribute | Description | Example Reference |
|-----------|-------------|-------------------|
| Premium | Generous whitespace, subtle elevation | Stripe |
| Confident | Bold CTAs, clear hierarchy | Vercel |
| Minimal | Limited palette, no gradients | Linear |

## We Avoid

| Attribute | Description | Why |
|-----------|-------------|-----|
| Playful | Bright colors, rounded everything | Off-brand |
| Dense | Packed information, tight spacing | Hurts readability |
| Decorative | Gradients, patterns, heavy borders | Visual noise |

## Key Tensions

When attributes conflict, use these priorities:

1. **Readability > Density** - always prioritize legibility
2. **Premium > Minimal** - spacious wins over tight
3. **Confident > Subtle** - for CTAs and key actions
