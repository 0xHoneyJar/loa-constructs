# Design References

## Primary Influences

### Stripe
- **URL**: https://stripe.com/docs
- **Screenshot**: moodboard/stripe-dashboard.png
- **What we like**:
  - Generous padding (p-6 to p-8)
  - Subtle shadows (shadow-sm)
  - Clean typography hierarchy
  - Confident blue CTAs
- **What we skip**:
  - Purple accent colors
  - Gradient backgrounds

### Linear
- **URL**: https://linear.app
- **Screenshot**: moodboard/linear-sidebar.png
- **What we like**:
  - Tight density in navigation
  - Monospace accents for labels
  - Dark mode execution
  - Keyboard-first UX indicators
- **What we skip**:
  - Icon-heavy navigation
  - Extreme density in main content

### Vercel
- **URL**: https://vercel.com/dashboard
- **Screenshot**: moodboard/vercel-buttons.png
- **What we like**:
  - Sharp corners on buttons
  - High contrast CTAs
  - Minimal color palette
  - Clear deployment status patterns
- **What we skip**:
  - Complex multi-step deployment UI

## Anti-References (What NOT to Do)

### Bootstrap
- **Why avoid**: Too rounded, too colorful, generic feel
- **Specific patterns to avoid**:
  - Default btn-primary blue
  - Card borders on everything
  - Generic form styling
  - Alert color palette

### Material Design (Older)
- **Why avoid**: Overly prescriptive, dated feel
- **Specific patterns to avoid**:
  - Floating action buttons
  - Heavy elevation shadows
  - Ripple effects everywhere
