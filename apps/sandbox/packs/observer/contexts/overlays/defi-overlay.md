## DeFi-Specific Context

### Terminology

| Term | Meaning | Signal Value |
|------|---------|--------------|
| APY | Annual Percentage Yield | Interest in returns |
| APR | Annual Percentage Rate | Often confused with APY |
| IL | Impermanent Loss | LP understanding |
| TVL | Total Value Locked | Protocol metrics awareness |
| Yield | Returns on deposits | Core motivation |

### Common Pain Points

- Gas cost unpredictability
- Complex multi-step transactions
- Impermanent loss confusion
- APY vs APR confusion
- Transaction timing sensitivity
