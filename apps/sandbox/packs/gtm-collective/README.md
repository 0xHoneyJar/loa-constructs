# GTM Collective

Go-To-Market skills and commands for product launches, positioning, and developer relations.

## Overview

The GTM Collective is a comprehensive suite of AI-powered skills and commands designed for product teams executing go-to-market strategies.

## Skills (8)

| Skill | Description |
|-------|-------------|
| `analyzing-market` | Market research, TAM/SAM/SOM, competitive analysis |
| `building-partnerships` | Partnership strategy and ecosystem development |
| `crafting-narratives` | Story development and messaging frameworks |
| `educating-developers` | DevRel strategy and developer content |
| `positioning-product` | Product positioning and differentiation |
| `pricing-strategist` | Pricing strategy and packaging |
| `reviewing-gtm` | GTM plan review and feedback |
| `translating-for-stakeholders` | Executive communication and stakeholder alignment |

## Commands (14)

### Setup & Configuration
- `/gtm-setup` - Initialize GTM workspace
- `/gtm-adopt` - Adopt existing GTM materials
- `/gtm-feature-requests` - Track feature requests

### Sync & Review
- `/sync-from-gtm` - Sync from GTM documentation
- `/review-gtm` - Review GTM plan
- `/sync-from-dev` - Sync from development

### Market Analysis
- `/analyze-market` - Analyze market landscape
- `/position` - Define product positioning
- `/price` - Develop pricing strategy

### Planning
- `/plan-partnerships` - Plan partnership strategy
- `/plan-devrel` - Plan developer relations
- `/plan-launch` - Plan product launch

### Communication
- `/create-deck` - Create presentation deck
- `/announce-release` - Announce product release

## Installation

```bash
# Via Loa Constructs CLI
loa install gtm-collective

# Or via /constructs command
/constructs
# Select "gtm-collective" from the list
```
